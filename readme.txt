== Description ==
Mint is a simple, blog-focused theme that can also be used as parent theme. Mint's simple, straightforward typography is readable on a wide variety of screen sizes, and suitable for multiple languages. The layout is responsive so the site will look good on all screen sizes from mobile to desktop.

* Responsive Layout
* Two different layout options - boxed or wide
* Page layouts
* Theme Customizer
* Theme Options
* Custom Sidebars
* Footer Widgets
* Social Icons
* Retina Ready
* Menu Description
* The GPL v3.0 or later license. :) Use it to make something cool.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's ZIP file. Click Install Now.
3. Click Activate to use your new theme right away.
